[WEB2 CSS - 1. 수업 소개]
왜 CSS가 생겨날 수 밖에 없었는지 알게 될 것이며 자주 사용되는 중요한 것들을 통계를 근거로
배우게 될 것이다. 그렇게 우리는 사이트를 아름답게 만들 수 있을 것이다.

[WEB2 CSS - 2. CSS가 등장하기 전의 상황]
HTML안에서 변경하도록 태그를 추가했었다고 한다. 그 중 하나가 <font>이다.

[WEB2 CSS - 3. CSS의 등장]
CSS가 나오면서 html에 적어주지만 CSS임을 알리기 위해 <head>안에 <style>을 적어 주었다. 
그러면서 하나의 값만 바꿔주면서 그 코드를 포함하는 모든 것이 바뀌게 된다.  이런 폭발적인 
효과가 생기므로 새로운 언어를 배워야하고 컴퓨터도 새로운 언어를 인식해야하지만 사용하는 
것이다. 디자인에 대한 것은 style에 전부 들어가게 하면서 즉 CSS를 사용하면서 html은 html에 
집중하게 하고 html에서 만든 디자인보다 나은 디자인을 하기 위해 CSS가 만들어졌다.

[WEB2 CSS - 4. CSS의 기본 문법]
<style>태그 안에 선택자가 있고 선택자의 중괄호 안에 그 선택자에게 지정될 효과를 적어준다.
style="color:red"라는 식의 html의 속성을 사용하여 CSS로 해석하도록 할 수도 있다. 이렇게
사용할 경우는 선택자를 사용할 필요가 없다. 
즉 두개의 경우가 있다 style태그를 사용하거나 style속성을 사용하는 방법이다.
색을 넣는 효과는 color:red 이렇게 사용하고 문자를 꾸며주는 것은 text-decoration:
이와 같이 사용한다. 효과를 적어줄 때는 문장의 마지막에 ;를 적어 구분한다. 만약 한 문장에만
사용하고 싶다면 그 문장에 style속성을 사용하여 예외를 둘 수 있다.

[WEB2 CSS - 5. 혁명적 변화]
selector         a {
Declaration        color:      red
                   }  Property   Value
의 형식으로 선택자를 사용하게 되고 모르는 것이 선택자인지 효과인지 알 수 있다.
그렇다면 선택자의 의미를 모른다면 검색할 수 있고 배울 수 있다.
