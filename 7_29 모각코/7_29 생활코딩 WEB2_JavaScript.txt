[WEB2 JavaScript - 1. 수업 소개]
웹이 생기고 인간은 욕심이 생깁니다. 사용자와 상호작용 하기를 원했고 그래서 
생겨난 것이 JavaScript이다.

[WEB2 JavaScript - 2. 수업의 목적]
JavaScript는 사용자와 상호작용하기 위한 언어이고 웹브라우저는 한번 바뀌면
스스로는 바꿀 수 없다. 즉 JavaScript를 이용하여 웹브라우저를 바꿀 수 있고
웹브라우저를 좀 더 동적으로 만들어준다. 예를 들면 버튼을 넣어 기능을
추가하는 것이 있다.

[WEB2 JavaScript - 3. HTML과 JS의 만남: script태그]
HTML과 JavaScript는 전혀 다른 언어이다. 이것을 어떻게 연결시킬 것인가. 이때
<script></script>라는 태그를 사용한다. 이 안에서는 HTML과 똑같이 글씨를
출력할 수 있지만 1+1을 넣는다면 2로 계산되어 2가 출력된다.
JS는 동적으로 동작하여 HTML과는 다르다는 것을 확인하였다.
<script></script>안에서의 출력은 document.write();를 사용하고 소괄호 안에
입력할 내용을 적어준다. 글자를 출력하고 싶다면 ""를 사용한다.

[WEB2 JavaScript - 4. HTML과 JS의 만남: 이벤트]
사용자와 상호작용하는 핵심적인 내용이다. 그 중 영상에선 버튼을 만들었다.
<input type:"button">을 사용하였고 버튼 안의 글씨는 value=""를 사용하였고
한 번 클릭 했을 때 무언가를 일어나게 하고 싶다면 onclick=""를 사용하였다. 
이런 웹브라우저 위에서 일어나는 일들을 사건 이벤트라고 한다. 즉 이런 이벤트를
일어나게 하는 기능중 하나가 버튼을 클릭하는 것이다. 또 다른 것은 
type="text"로 바꾼다면 그 안에 글씨를 입력할 수 있게 되는데 onchange=""를 
사용하여 창안의 글씨가 바뀌었다면 이벤트가 일어나게 할 수 있고, onkeydown
이라는 이벤트는 창안에 커서가 있고 키보드의 키가 눌렸을 경우 이벤트가
일어나도록 만들 수 있다. 

[WEB2 JavaScript - 5. HTML과 JS의 만남(콘솔)]
지금까지는 웹사이트에서의 작업은 HTML을 작성했다면 이번엔 웹사이트에 검사안에
console을 이용해본다. 짧게 무언가를 동작하게 할 때 예를 들어 글자 수를 알고
싶다면 글자를 복붙하고 ''로 감싸고 alert의 소괄호 안에 넣고 .length를 붙이면
즉 alert(''.length)이렇게 한다면 경고창이 뜨면서 ''안의 문자의 글자수를 
경고창에 보여줄 것이다.
이렇게 JS를 사용한다면 웹사이트를 만드는 것 말고도 이미 만들어져있는
웹사이트를 나의 필요에 맞게 사용하는 곳에도 쓰인다는 것을 알 수 있다.

[WEB2 JavaScript - 6. 데이터타입-문자열과 숫자]
6개의 타입중 그나마 친근한 문자열과 숫자를 먼저 알아본다. 검사의 콘솔창을
열면 숫자를 입력하거나 문자를 입력하면 그에 맞는 결과가 나오고
JS에 문법을 사용하여 .length나 .toUpperCase등을 사용하면 그에 맞는 결과를
알려준다.

[WEB2 JavaScript - 7. 변수와 대입 연산자]
이 영상에서는 전 영상과 같이 java 언어에서의 내용과 같은 내용을 설명한다.
변수와 변수에 값을 대입한다. 그후 그 변수의 연산을 수행한다. 상수에는 상수를
대입하지 못한다. 변수를 사용할 때는 var을 앞에 적어주고 변수의 이름을 
적어준다.

[WEB2 JavaScript - 8. 웹브라우저 제어]
직접<body>태그에 style을 넣어 직접 바꿀 수도 있지만 언제나 이 디자인으로
고정된다. 그러므로 JS를 이용하여 사용자의 입력에 따라 바뀔 수 있게 만들어야
한다. 그러기 위해서는 CSS에 대해 조금 알아야한다.

[WEB2 JavaScript - 9. CSS 기초: style 속성]
CSS를 안다면 볼 필요가 없다고 한다. style속성을 이용하여 기초적인 텍스트나 
배경의 색변화 등을 알려준다.

[WEB2 JavaScript - 10. CSS 기초 (style태그)]
CSS에서 배운 것을 알려주고 태그를 이용하여 같은 클래스나 id를 가진것을 
한번에 변경하는 법을 알려줬다.
