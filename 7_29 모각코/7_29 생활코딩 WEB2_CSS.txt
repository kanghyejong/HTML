[WEB2 CSS - 12. 미디어 쿼리 소개]
모든 기계에서 보여져야 할 웹을 위하여 그에 대응하는 반응형 웹디자인을 사용하였다. 
반응형 디자인은 웹의 크기에 따라 웹이 보여지는 형태가 달라지는 것이다.
그 중 핵심적인 내용인 미디어 쿼리를 사용한다.
기본적으로 @media(조건){}의 형태를 띄고 조건의 예시로 화면의 크기가 최소 
800px인 것을 사용한다면 @media(min-width:800px){}를 사용할 수 있다.

[WEB2 CSS - 13. 미디어 쿼리 써먹기]
이전에 만들었던 웹을 크기가 800보다 작다면 가독성을 높이기 위해 선을 없애고 
컬럼으로 나눠지지 않도록 하려고 한다. 이때 @media(max-width:800px){}를 
사용하고 중괄호 안에 컬럼으로 나눠지지 않고(display:block) 
선을 없애며 (border:none); 등을 사용하여 기존의 CSS에 대해 다른 값이 적용되도록
한다.  

[WEB2 CSS - 14. CSS코드의 재사용]
이제 만들었던 코드들을 다른 파일들에도 적용을 시켜줘야 한다. 이때 중복을 
없애기 위해 style.css 타입의 파일로 만들어주고 이것을 html에 link를 사용하여
추가해준다. <link rel="stylesheet" href="style.css"> 이로인해 웹사이트에
똑같은 변경이나 추가에 대해 style파일에서 한 번의 수정만으로 사이트의 수가 
몇천 몇억이 될지도 모르는 상황에 대한 걱정에서 벗어나게 해주는 혁명적인
변화를 가져왔다. 또한 html자체에서도 코드길이가 줄고 가독성도 좋아지며
style.css 파일이 어떻게 코딩되어있는지 몰라서 사용할 수 있게 되었다. 한 번 
들어갔다면 style.css파일이 캐시로 남아 네트워크를 사용하지 않게 되어 
웹사이트가 더 빨리 열리고 코드가 짧아지면서 인터넷 사용료도 적게 들 것이다.

[WEB2 CSS - 15. 수업을 마치며]
이제 배운 것으로 웹사이트를 꾸미면서 스스로 공부하고 발전해나가야 할 것이다.
